- 最新提交：见 GitHub main 最新提交

## 已完成

### v0.1 基础可玩版

- 创建角色、选择出身、男女立绘入口。
- 年龄、寿元、境界、修为、气血、心境、悟道等基础属性。
- 静室修炼、外出历练、秘境探险、闭关、结交同道、尝试突破。
- 修行纪年、本地存档、转世重开。

### v0.2 移动端与交易版

- 移动端页签布局，行动与纪年放在同一主界面。
- 状态精简展示，详细状态改为弹窗。
- 行囊、坊市、交易行合并。
- 货品、丹药、破境丹可单独出售，不再一键清空。
- 获得物品和灵石时弹出提示。

### v0.3 地图与破境筹备版

- 山海地图与不同区域危险度。
- 灵草、妖兽内丹、皮骨、妖骨、雷纹晶、五行玉、残卷等资源。
- 破境不只看修为，加入心境、悟道、丹药、材料、筹备加成。
- 筑基丹、结金丹、化婴丹等破境丹接入突破成功率。
- 青岚宗、血河路线、散修身份雏形。

### v0.4 三层随机事件版

- 三层随机事件生成：
  - 第一层：事件骨架。
  - 第二层：地点、人物、冲突、隐藏暗线、局势压力。
  - 第三层：选项、结果、副作用、奖励、死亡风险。
- 外出、秘境、论道、宗门任务、药园、小比、魔窟、黑市、夺宝接入随机事件。
- 同名任务也会随机出不同路径，开发时也无法完全预测具体走向。
- GitHub 发布链路打通，可从本地提交并推送到线上。

### v0.5 门派体系大改

- 青岚宗不再只是稳定温室，加入宗门规矩、派系、人情、温室滞碍。
- 血河不再只是杀人夺宝，加入魔门秩序、旧约、同门竞争、血债。
- 正道和魔道不做绝对好坏，事件里会出现规矩、人情、底线和灰度选择。
- 增加宗门信任、派系牵连、因果、温室滞碍、血债等长期变量。
- 身份显示会根据门派信任、贡献、声望变化为外门骨干、内门弟子、血令弟子等。
- 世界事件会根据派系牵连、温室滞碍、血债、通缉等变量触发门派压力。

### v0.6 道心与长期后果

- 增加道心路线：守正、利己、杀伐、求真、无情。
- 选择会推动道心倾向，并显示在身份中。
- 增加人情、仇怨、心魔、暗伤等长期变量。
- 人情、仇怨、心魔、暗伤会在后续世界事件中回流。
- 正邪声望、通缉、尘缘、因果共同影响世界对玩家的态度。

### v0.7 破境大改第二版

- 境界越高突破越难，成功率上限明显降低。
- 破境受历劫、因果、心魔、暗伤、根基瑕疵影响。
- 高境界不能只靠丹药硬冲，材料、历劫、道心隐患都会参与计算。
- 失败不只是扣血，会留下暗伤、心魔、根基瑕疵。
- 成功破境会洗去部分旧劫和隐患。

### v0.8 丹铺与品质购买

- 丹铺改为玩家自选购买，不再由系统随机替玩家买丹。
- 坊市每年刷新货架，常用丹和破境丹都有库存，买完会减少。
- 破境丹增加下品、正品、上品、极品品质，品质越高越贵、越少见。
- 普通坊市主要出现低品质和正品，上品极少，极品主要靠副本、黑市、奇遇、拍卖机缘。
- 破境丹显示目标境界、品质、加成和价格，玩家可以按自己资源选择是否购买。

### v0.8 丹药、炼丹与天材地宝

- 加入炼丹或委托炼丹。
- 炼丹会消耗灵草、妖兽皮骨、妖兽内丹、灵矿、五行玉等材料。
- 炼丹加入熟练度、成丹率、失败损耗、丹毒积累。
- 天材地宝不只出售，也能用于突破、疗伤、炼器、宗门贡献。
- 高阶丹药获取更难，需要地图、宗门、黑市、机缘共同支撑。

### v0.9 功法与战斗生态

- 功法品阶、属性倾向、修炼路线。
- 功法影响修炼速度、战斗收益、突破准备和事件选项。
- 妖兽增加种类、生态、掉落、危险特性。
- 副本增加队伍、团灭、撤退、分赃、背刺、救援等机制。

### v1.0 本地完整试玩版

- 加入本地备份码，可复制保存当前存档。
- 加入访客码、本机打开次数、最近战报。
- 由于当前仍是 GitHub Pages 静态页，真实在线人数、云存档、排行榜需要后端或第三方统计服务。
- 版本号在游戏界面可见。
- changelog 通过 VERSION.md 维护。
- 适合朋友长期试玩的稳定版本。
