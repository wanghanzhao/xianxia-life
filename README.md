# 尘缘修仙录

原创修仙类文字模拟人生原型。

## 玩法

- 创建角色并选择出身。
- 通过静室修炼、外出历练、闭关、结交同道推进人生。
- 修为达标后尝试突破境界。
- 寿元、气血、心境会影响结局。
- 浏览器本地存档，点击“转世”可重开。

## 验收标准

- 打开 `index.html` 可直接游玩。
- 至少包含 4 种行动：修炼、历练、闭关、社交、突破。
- 至少包含 6 个境界阶段。
- 角色数据可保存到 `localStorage`。
- 寿元耗尽或气血归零后进入结局状态。

## 回滚方案

删除 `xianxia-life` 目录即可移除本项目，不影响其他项目文件。

## 发布

这是纯静态网页项目，仓库根目录包含：

```
index.html
styles.css
src/game.js
```

可通过 GitHub Pages 发布：`Settings -> Pages -> Deploy from branch -> main / root`。
